# Task 7 – Basic Sales Summary using SQLite & Python

## 📌 Objective
Use SQL inside Python to extract simple sales information and display results using print statements and a basic bar chart.

---

## 🛠 Tools Used
- Python  
- sqlite3  
- pandas  
- matplotlib  
- Google Colab  

---

## 📊 Project Overview
This project demonstrates how to integrate SQLite with Python to perform basic sales analysis.

The notebook performs the following steps:

1. Creates a small SQLite database (`sales_data.db`)
2. Creates a `sales` table
3. Inserts sample sales data
4. Executes SQL queries to calculate:
   - Total quantity sold
   - Total revenue per product
5. Loads SQL query results into a pandas DataFrame
6. Displays results using:
   - Printed DataFrame
   - Bar chart (Revenue by Product)

---

## 📁 Files Included
- `Task-7.ipynb` – Jupyter Notebook with complete implementation  
- `sales_data.db` – SQLite database file  
- `sales_chart.png` – Revenue bar chart (optional export)

---
✅ Learning Outcomes

1. Connect SQLite database in Python
2. Execute SQL queries using sqlite3
3. Load SQL results into pandas
4. Perform aggregation using GROUP BY
5. Visualize sales data using matplotlib
